# License-Plate-Recognition-Using-K-Nearest-Neighbors

## Project Description

This project was developed as part of a Machine Learning course assignment.

The objective of this project is to recognize characters from segmented license plate images using the K-Nearest Neighbors (KNN) algorithm.

License plate recognition is a widely used computer vision application in intelligent transportation systems. This project focuses on the character recognition stage, where pre-segmented character images are classified into their corresponding letters and digits.

The project includes image preprocessing, feature extraction, KNN model training, and performance evaluation, providing hands-on experience with image classification using traditional machine learning techniques.

## Dataset

The dataset was provided by the course instructor for educational purposes.

- Task: License plate character recognition
- Model: K-Nearest Neighbors (KNN)
